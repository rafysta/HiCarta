    使い方¶

    「やりたいこと」から探せる、最短の操作ガイドです。細かいオプションの意味は、各項目からリンクしている 画面と操作の説明 を参照してください。

    まずは大まかな流れだけ覚えておけば十分です。データを読み込む → マップを開く → 移動して見る、これだけです。

    データを読み込む¶

    データの読み込みは、画面上部の データ ボタンを押すと開く読み込み画面（「Hi-C」「トラック」「セッション」の 3 タブ）から行います。

    Hi-C マップを読み込む¶

    1. 上部の データ ボタンをクリックします。
    2. Hi-C タブで、メニューを読み込む を押してサンプルとデータセットを選ぶか、ローカル .hic ファイル にパソコン内の .hic のパスを入れます（参照… から選択も可）。
    3. マップを開く をクリックします。

    → 詳細: 画面と操作の説明 - データ

    bigWig を読み込む¶

    1. 先に Hi-C マップを開いておきます（トラックはマップの座標に合わせて描かれます）。
    2. データ ボタン → トラック タブを開きます。
    3. bigWig/BED ファイル/URL に bigWig のパスまたは URL を入れます（参照… も可）。
    4. 種類 を「bigWig」にして、トラックを追加 をクリックします。

    → 詳細: 画面と操作の説明 - トラック

    Border Strength を読み込む¶

    1. Hi-C マップを開いておきます。
    2. データ ボタン → トラック タブを開きます。
    3. ファイル欄に Border Strength ファイル（*_BS.txt）のパスまたは URL を入れます。
    4. 種類 を「境界強度（Border Strength）」にして、トラックを追加 をクリックします。

    正の値は赤、負の値は青、境界の位置には破線が引かれます。ファイル形式は データ形式 を参照してください。

    → 詳細: 画面と操作の説明 - トラック

    遺伝子・BED を読み込む¶

    トラック タブで、ファイル欄にパスまたは URL を入れ、種類 を「遺伝子（GFF3）」または「BED」にして トラックを追加 をクリックします。IGV XML やトラックリストが用意されている場合は、上部の 読み込む から XML ファイル → カテゴリ → トラックの順に選んで追加することもできます。

    → 詳細: 画面と操作の説明 - トラック

    表示する領域を移動する¶

    移動の操作は、画面左の 移動 メニューにまとまっています。

    任意の場所に移動する¶

    1. 上部メニューの 移動 を選びます。
    2. 染色体 を選び、Y 軸 開始 と Y 軸 終了（bp）に見たい範囲を入力します。
    3. 領域へ移動 をクリックします。

    → 詳細: 画面と操作の説明 - 移動

    ボタンで移動する¶

    移動 メニューの方向パッド（上下左右＋斜めの 8 方向）で、マップを動かせます。1 回の移動量は 移動量（¼・½・1 画面分）で選べます。マップ上を直接ドラッグしたり、スクロールでズームしたりもできます。

    → 詳細: 画面と操作の説明 - 移動

    染色体全体を表示する¶

    移動 メニューの方向パッド中央のボタン（⌂）をクリックすると、その染色体の全体表示に戻ります。

    → 詳細: 画面と操作の説明 - 移動

    気になる場所をブックマークする¶

    1. ブックマークしたい領域を表示した状態にします。
    2. 移動 メニュー下部の名前欄に、必要なら名前を入力します（省略可）。
    3. ★ この表示を保存 をクリックします。

    保存したブックマークは一覧に並び、クリックするといつでもその表示に戻れます。不要になったら 削除 で消せます。

    → 詳細: 画面と操作の説明 - 移動

    見た目を調整する¶

    色の濃さ（コントラスト）を変える¶

    上部メニューの 表示 を選び、最大値 のスライダーを動かすとコントラストが変わります。線形／log10 の切り替えや、カラーパレット の選択もここで行います。

    → 詳細: 画面と操作の説明 - 表示

    マップの高さやレイアウトを整える¶

    上部メニューの 設定 で、コンタクトマップの高さ を指定したり、ウィンドウに合わせる で画面いっぱいに広げたり、トラックの高さや解像度を調整したりできます。

    → 詳細: 画面と操作の説明 - 設定

    保存・出力する¶

    表示状態を保存して後で再現する（セッション）¶

    保存する:

    1. 上部の データ ボタン → セッション タブを開きます。
    2. 現在の表示を保存 をクリックすると、.json ファイルがダウンロードされます。

    復元する:

    1. データ ボタン → セッション タブを開きます。
    2. ファイルから復元（.json） で、保存しておいた .json を選びます。

    データ元・領域・カラースケール・すべてのトラックがまとめて復元されます。

    → 詳細: 画面と操作の説明 - データ（セッション）

    画像として出力する・印刷する¶

    1. 出力したいマップを開いておきます。
    2. 上部メニューの 印刷 を選び、印刷プレビューを開く をクリックします。
    3. プレビュー画面で、送信先（プリンター／ファイル）、フォーマット（PNG／PDF）、用紙サイズ、出力領域、座標目盛・凡例・余白の有無、トラックを含めるかを設定します。
    4. 実行 をクリックします。

    → 詳細: 画面と操作の説明 - 印刷

    その他¶

    表示言語を切り替える¶

    上部メニューの 設定 → 設定ファイルを編集… を開き、表示言語 を選んで 設定を反映 をクリックします。ページが再読み込みされ、言語が切り替わります。このとき、開いているマップやトラックは閉じられるので、同じ状態に戻したい場合は セッションの保存機能 を使ってください。

    → 詳細: 画面と操作の説明 - 設定
