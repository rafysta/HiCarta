    画面と操作の説明¶

    このページは、HiCarta の各メニューと項目が何をするものかを詳しく説明するリファレンスです。「まず何をすればよいか」を知りたい場合は 使い方 を先にご覧ください。

    画面上部には次のメニューがあります: データ · 移動 · 表示 · トラック · 印刷 · 設定 · 情報。データ ボタンを押すと読み込み画面が開き、その他のメニューは画面左の操作パネルを切り替えます。

    典型的な流れは、データを選ぶ → 領域を設定する → マップを開く → 移動・ズーム → トラックを追加、です。

    データ（読み込み画面）¶

    上部の データ ボタンを押すと、読み込み画面が開きます。Hi-C / トラック / セッション の 3 つのタブがあります。

    Hi-C タブ¶

    表示するコンタクトマップを選びます。方法は 2 通りです。

    メニューから選ぶ — メニューを読み込む をクリックすると、config.txt の menu_url で指定した Juicer メニューを取得し、サンプル と データセット を選べます。ドロップダウンは部分一致検索に対応しており、名前の一部を入力して絞り込めます。正規化 も選択できます。

    ローカル .hic ファイルを指定する — パソコン内の .hic のパスを .hic ファイルパス に貼り付けるか、参照… から選びます。ローカルファイルの指定はメニューの選択より優先されます。

    選び終えたら マップを開く をクリックすると読み込み・表示されます。

    Note

    サンプルメニュー内の各 .hic は単一解像度のファイルの場合があります。HiCarta は要求された解像度・正規化を、そのファイルが実際に持つものに合わせて調整します。

    トラックタブ¶

    → 下の「トラック」の節 を参照してください（読み込み画面のトラックタブと、左メニューのトラックは同じ設定を扱います）。

    セッションタブ¶

    表示全体をファイルに保存・復元します。

    現在の表示を保存 — データ元・領域・カラースケール・すべてのトラックを含む表示状態を、.json ファイルとしてダウンロードします（ファイル名は HiCarta_session_日時.json）。

    ファイルから復元（.json） — 保存しておいた .json を選ぶと、その表示を丸ごと再現します。

    移動¶

    表示する染色体と領域を指定し、マップ上を移動します。

    染色体 — 表示する染色体を選びます。染色体名は .hic の内部名に従います（例: S. pombe では I / II / III）。別の染色体を選ぶと、マップは自動的に開き直されます。

    Y 軸 開始 / Y 軸 終了（bp） — 表示する範囲を bp で指定します。

    領域へ移動 — 入力した染色体・範囲へジャンプします。

    画面移動（方向パッド） — 上下左右＋斜めの 8 方向ボタンで、表示中の領域を動かします。上下は Y 軸方向、左右は X 軸方向の移動です。中央の ⌂（ホーム）ボタンで、その染色体の全体表示に戻ります。

    移動量 — 方向ボタン 1 回あたりの移動量を、表示範囲に対する割合（¼ · ½ · 1）で選びます。

    ブックマーク — 今見ている表示を保存し、あとから戻れます。名前欄（省略可）に名前を入れ、★ この表示を保存 で登録します。登録した表示は一覧に並び、クリックでその場所へジャンプ、削除 で消去します。ブックマークはセッションにも保存されます。

    表示¶

    カラースケールを調整します。色は全タイル共通の 1 つの絶対値スケール を使うため、タイルの継ぎ目が出ません。

    カラーパレット — 配色を選びます（matlab, gentle, red, blue の 4 種類）。

    最大値（Max） — このスライダーを動かすとコントラストが変わり、すべてのタイルが再描画されます。初期値は、マップを開いたときに全体を粗く走査して自動で推定されます。

    線形 / log10(値) — 値スケールの取り方を切り替えます。最小値は 0 に固定されています。

    トラック¶

    コンタクトマップの下に 1 次元トラックを重ねて表示します。トラックはマップの横方向の移動・ズームに追従し、表示中の範囲だけを読み込みます。複数のトラックを、個別の色と高さで積み重ねられます。カーソルのガイド線がトラックを貫いて表示されます。

    トラックの追加は、読み込み画面（データ ボタン）の トラック タブから行います。

    ファイルを直接指定する — bigWig/BED ファイル/URL にパスまたは URL を入れ（参照… も可）、種類 を選び、必要なら ラベル・色・高さ（px） を設定して トラックを追加 をクリックします。すべて消去 で全トラックを削除します。

    IGV XML / トラックリストから選ぶ — 読み込む でトラックリストや IGV XML を読み込み、XML ファイル → カテゴリ → トラック の順に選んで追加します。

    扱えるトラックの種類:

    種類 表示内容 備考
    bigWig 塗りつぶし面 rtracklayer で読み込み
    BED 区間のボックス rtracklayer で読み込み
    遺伝子（GFF3） 遺伝子の矢印・エクソン・名前 ＋鎖は上／−鎖は下。ズームインでエクソン表示。名前は重ならないよう間引き。<gff3>.genes.rds にキャッシュ
    境界強度（Border Strength） BS.norm の面 正＝赤、負＝青、基準線 0。境界に破線

    config.txt の track_list_url は、単一の IGV XML か、複数の IGV XML の URL を 1 行ずつ並べた索引ファイルを指せます。各形式の詳細は データ形式 を参照してください。

    印刷¶

    現在開いているマップの指定領域を、画像または PDF として出力したり、印刷したりします。

    印刷プレビューを開く をクリックすると、プレビュー画面が開きます。プレビューでは次を設定できます。

    • 送信先 — プリンター、またはファイル。
    • 出力フォルダ / ファイル名 / フォーマット — ファイル出力時の保存先とファイル名、フォーマット（画像（PNG） または PDF）。
    • 用紙サイズ — A4 縦・A4 横・正方形・カスタム（幅・高さを mm で指定）。
    • 出力領域 — 染色体・開始・終了（bp）。Hi-C では縦・横は同一範囲になります。
    • 座標目盛を入れる / 凡例を入れる / 余白を空けない — 出力に含める要素の切り替え。
    • トラックも出力する — 表示中のトラックを一緒に出力します。

    設定したら 実行 をクリックします。

    設定¶

    レイアウトとアプリの既定値を調整します。

    コンタクトマップの高さ（px） — マップの高さを指定します。適用 で反映します。

    ウィンドウに合わせる — マップの高さを「ウィンドウの高さ − トラック高さの合計」に合わせます。

    トラック解像度（表示分割数） / 自動調整 — トラックの描画解像度や高さを調整します。

    集計方法（染色体全体表示） — 全体表示のときの集計を、平均（IGV 既定） か 最大（ピーク保持） から選びます。

    設定ファイルを編集…¶

    設定ファイルを編集… を押すと、アプリ起動時の既定値（config.txt）を確認・編集できます。

    • 表示言語 — 日本語 / 英語を切り替えます。
    • 既定の Juicer メニュー URL — 起動時に読み込むメニュー。
    • 既定のトラックリスト / IGV XML URL — トラックの既定リスト。

    設定を反映 をクリックすると、ページが再読み込みされて設定が適用されます（このとき、現在読み込んでいるマップやトラックは閉じられます）。

    情報¶

    バージョンと作者情報（谷澤 英樹、rafysta@gmail.com）を表示します。

    マップ上での操作¶

    マップを開いたあとは、マップ上で直接操作できます。

    • ドラッグ で移動
    • スクロール でズーム（解像度は自動で切り替わります）
    • 中クリックのドラッグ で範囲指定のズーム（ラバーバンド）

    ヘッダーには現在の座標と解像度が表示され、端には目盛り付きのルーラーが出ます。カーソルを合わせると、その位置の座標・スコア・距離が表示されます。座標系は左上が原点で、ゲノム座標の y は下向きに増加します。

    注意と対応範囲¶

    • HiCarta は現在、cis（同一染色体内）のコンタクトを対象としています。
    • トラックの染色体名（例: chrII）は、可能な範囲で .hic の名前（II）に対応付けられます。
