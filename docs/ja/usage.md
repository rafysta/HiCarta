# 使い方

上部メニューには6つのパネルがあります: **Data（データ）· Region（領域）· Display（表示）· Tracks（トラック）· Setting（設定）· About（情報）**。典型的な流れは、データを選ぶ → 領域を設定する → **Open map** → 移動／ズーム → トラックを追加、です。

## 1. Data（データ）

表示するコンタクトマップを選びます。方法は2つあります。

- **メニュー** — **Load** をクリックして `config.txt` の `menu_url` で指定した Juicer メニューを取得し、**Sample** と **Dataset** を選びます。ドロップダウンは部分一致検索に対応しており、名前の一部を入力して絞り込めます。
- **ローカル `.hic` ファイル** — マシン上の `.hic` のパスを貼り付けます。ローカルファイルはメニューの選択より優先されます。

サンプルメニュー内の各 `.hic` は単一解像度のファイルです。HiCarta は要求された解像度／正規化を、ファイルが実際に持つものに合わせて調整します。

## 2. Region（領域）

表示する**染色体**と **Y軸の範囲**を設定します。**Go to region** でジャンプします。別の染色体を選ぶと、マップは自動的に開き直されます。染色体名は `.hic` に従います（例: *S. pombe* では `I / II / III`）。

## 3. Open map（マップを開く）

**Open map** をクリックして読み込み・表示します。

- **ドラッグ**で移動
- **スクロール**でズーム（解像度は自動で切り替わります — レベル・オブ・ディテール）
- **中クリックのドラッグ**で範囲指定のズーム（ラバーバンド）

ヘッダーには現在の座標と解像度が表示され、端には補助目盛り付きのルーラーが表示されます。原点は左上で、ゲノム座標の *y* は下向きに増加します。

## 4. Display（表示）

カラースケールを制御します。色はすべてのタイルで共有される**1つの絶対値スケール**を使います（そのため継ぎ目がありません）。**Max** を調整するとコントラストが変わり、すべてのタイルが再描画されます。初期の `Max` は、Open 時に全マップを粗く走査して推定されます。

## 5. Tracks（トラック）

コンタクトマップの下に1次元トラックを追加します。トラックはマップの横方向の移動／ズームに追従し、表示中の範囲だけを読み込みます。複数のトラックを、個別の色と高さで積み重ねられます。カーソルのガイド線がトラックを貫いて表示されます。

| 種類 | 表示内容 | 備考 |
|---|---|---|
| **bigWig** | 塗りつぶし面 | `rtracklayer` で読み込み |
| **BED** | 区間のボックス | `rtracklayer` で読み込み |
| **gene（GFF3）** | 遺伝子の矢印、エクソン、名前 | + 鎖は上／− 鎖は下；ズームインするとエクソンが表示；名前は重ならないよう間引き；`<gff3>.genes.rds` をキャッシュ |
| **Border Strength** | `BS.norm` の面 | 正 = 赤、負 = 青、基準線 0；境界に破線 |

`config.txt` の `track_list_url` は、単一の IGV XML、または複数の IGV XML URL を1行ずつ列挙したインデックスファイルを指せます。インデックスの場合は **XML file → Category → Track** の順に選びます。

## 6. Setting（設定）

レイアウトの調整: **map height（マップの高さ）**、**Fit to window**（マップ = ウィンドウの高さ − トラック高さの合計）、自動調整、トラック解像度。

## 注意と現在の対応範囲

- HiCarta は現在、**cis**（同一染色体内）のコンタクトを対象としています。
- `chrII` のようなトラックの染色体名は、可能な範囲で `.hic` の名前（`II`）に対応付けられます。
